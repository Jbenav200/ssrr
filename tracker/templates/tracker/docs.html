MAP

The focal point of our application. Here you can view the meteorite data visualized on a world map. Each meteorite impact is represented by a colored dot. You can click each dot to view more information about that specific meteorite. Use the filter button in the top left corner to only display meteorites of a chosen mass or on a selected continent. Use the globe button in the top left corner to go to the globe page. To open the main menu, click the burger bar in the top left corner.

MAP API

To visualise data on an interactive map we used the MapBox API. MapBox allows us to use JSON to create a feature layer, with each recorded meteorite as a separate feature. It then allows us to filter through the features and display them accordingly, changing the colour for each feature point (Dot on the map). MapBox was a solid choice due to its reputation, it has been used across many different web applications and is used by many big name brands to visualise data or even simply place a map on their websites.

DJANGO

Django is a powerful RESTful web framework for Python, with out of-the-box authentication and basic security measures which protect from Cross Site Scripting attacks, SQL Injection amongst others. It also s upports a CMS backend which we can use to import our data from NASA.

GLOBE API

ArcGIS is a web API that allows us to model a 3D globe on our website and use FeatureLayers to visualise our data. It also allows us to control how the data is visualised, in this case we visualised the data by mass and changed the point size and colour.
